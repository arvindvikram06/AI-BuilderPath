# NeuraStack Technologies — Product Catalog

## Product Overview
NeuraStack offers three AI-powered products designed for engineering teams: **NeuraSearch**, **NeuraChat**, and **NeuraPipeline**. Each product is available in three tiers: Free, Pro, and Enterprise.

---

## Product 1: NeuraSearch
**Description:** AI-powered semantic search engine for internal codebases, documentation, and knowledge bases.

### Features
- Natural language search across 50+ file formats (PDF, Markdown, DOCX, code files)
- Semantic similarity search using vector embeddings
- Integration with GitHub, Confluence, Notion, and Google Drive
- Search history and saved queries
- Role-based access control (RBAC) for document visibility

## Product 2: NeuraChat
**Description:** RAG-powered internal chatbot that answers questions from your company's documents and knowledge base.

### Features
- Multi-document ingestion (PDF, Markdown, DOCX, TXT)
- Conversational memory (up to 20 turns per session)
- Source citations for every answer
- Confidence scores per retrieved chunk
- Streamlit-based web interface
- REST API for integration into existing tools
- LLM backend switching: Groq, Azure OpenAI, Claude, Ollama
